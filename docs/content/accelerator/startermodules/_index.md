Starter modules provide opinionated implementations of Azure landing zone using Bicep or Terraform.

Update these modules as your organization's needs evolve. Each module expects different inputs, detailed on the following pages.

## Bicep Starter Modules

- **[Bicep Azure Verified Modules for Platform landing zone (ALZ)]({{< relref "bicep-platform-landing-zone" >}})**: Latest framework using Azure Verified Modules (iac_type: `bicep`)

## Terraform Starter Modules

- **[Terraform Azure Verified Modules for Platform landing zone (ALZ)]({{< relref "terraform-platform-landing-zone" >}})**: Management groups, policies, hub networking with fully custom configuration.
